# React Weather App 🌤️

A modern and responsive weather application built with React that provides real-time weather information for cities around the world. The application allows users to search for locations and view current weather conditions, temperature, humidity, wind speed, and other weather details through an intuitive user interface.

---

## Features

* Search weather by city name
* Current temperature display
* Weather condition icons
* Humidity information
* Wind speed details
* Responsive design for desktop and mobile devices
* Clean and modern user interface
* Error handling for invalid city names
* Fast and lightweight performance

---

## Technologies Used

* React.js
* JavaScript (ES6+)
* CSS3
* HTML5
* Weather API
* Vite

---

## Installation

### Clone the repository

```bash
git clone https://github.com/ritujane78/react_weather_app.git
```

### Navigate to the project directory

```bash
cd react_weather_app
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```

---

## Environment Variables

Create a `.env` file in the root directory and add your Weather API key:

```env
VITE_WEATHER_API_KEY=your_api_key_here
```

---

##  Usage

1. Launch the application.
2. Enter a city name in the search field.
3. Press Enter or click the search button.
4. View current weather information for the selected city.
